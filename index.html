¿Qué es Python?

Python es un lenguaje de programación simple y poderoso. Su sintaxis clara ayuda a concentrarte en las ideas sin pelearte con la forma. Se usa en desarrollo web, ciencia de datos, automatización, inteligencia artificial y más.

Tu primer programa

print("Hola, mundo")
print(objeto1, objeto2, ..., sep=" ", end="\n")

Caracteres especiales en cadenas

En Python, dentro de un texto (string) podés usar secuencias de escape que empiezan con una barra invertida \. Estas secuencias no se muestran como texto literal, sino que producen un efecto especial.

print("Hola\nMundo")
print("Uno\tDos\tTres")
print("Comillas: \"texto entre comillas\"")
"\\n" → salto de línea
"\\t" → tabulación (espacio grande)
"\\\"" → comillas dobles dentro de un string
"\\'" → comillas simples dentro de un string
"\\\\" → barra invertida literal
Explicación:
- \n inserta un salto de línea (como apretar Enter).
- \t agrega un tabulador (espacio ancho).
- \" y \' permiten escribir comillas dentro de un texto sin cerrar la cadena.
- \\ muestra una barra invertida real.

Ejemplo:
print("Nombre:\tAna\nEdad:\t25")
Resultado:
Nombre: Ana
Edad: 25
Parámetros:
- objeto1, objeto2...: valores a mostrar (texto, números, listas, etc.).
- sep: separador entre objetos (por defecto espacio).
- end: texto al final (por defecto salto de línea).

Ejemplo: print("Hola", "Mundo", sep="-") → Hola-Mundo.

¿Qué es una variable?

nombre = "Lucía"
edad = 20
altura = 1.65
es_estudiante = True
nombre_variable = valor
Una variable es un nombre que apunta a un valor en memoria. Python infiere el tipo a partir del valor. Tipos comunes: str (texto), int (entero), float (decimal), bool (True/False).

Buenas prácticas de nombres

  • Descriptivos: edad_usuario mejor que x.
  • Usar guion bajo para separar palabras: total_mensual.
  • Sin espacios ni símbolos raros; no empezar con números.
  • Distingue mayúsculas y minúsculas: Edad ≠ edad.

Conversión de tipos (casting)

numero_texto = "100"
numero = int(numero_texto)
print(numero + 50) # 150
int(valor) | float(valor) | str(valor) | bool(valor)
Parámetros: valor: dato a convertir.
int("42") → 42, float("3.14") → 3.14, str(42) → "42", bool(0) → False.
Si el contenido no representa el tipo, produce error (int("hola")).

Operaciones aritméticas básicas

a = 20; b = 6
print("Suma:", a + b)
print("Resta:", a - b)
print("Multiplicación:", a * b)
print("División real:", a / b)
print("División entera:", a // b)
print("Módulo:", a % b)
print("Potencia:", a ** 2)
a + b | a - b | a * b | a / b | a // b | a % b | a ** b
/ devuelve decimales; // trunca hacia abajo; % devuelve el resto; ** eleva a potencia.

Precedencia y paréntesis

print(2 + 3 * 4) # 14
print((2 + 3) * 4) # 20
Orden: ** → * / // % → + -. Usá paréntesis para expresar claramente la intención.

Divisiones: real, entera y módulo

Comparación práctica

print(7 / 2) # 3.5
print(7 // 2) # 3
print(-7 // 2) # -4
La división entera // redondea hacia abajo (piso), incluso con negativos.

Uso del módulo %

n = 42
print(n % 2 == 0) # True → es par

dia = 15
print(dia % 7) # 1 → mismo resto que 1, 8, 15...
% sirve para periodicidades (semanas, ciclos) y comprobaciones como par/impar.

Máximos, mínimos, sumas y redondeos

numeros = [10, 4, 27, 15, 27, -3]
print("Máximo:", max(numeros))
print("Mínimo:", min(numeros))
print("Suma:", sum(numeros))
print("Cantidad:", len(numeros))
promedio = sum(numeros) / len(numeros)
print("Promedio:", promedio)
print(round(3.14159, 2)) # 3.14
max(iterable) | min(iterable) | sum(iterable) | len(iterable) | round(numero, ndigits=0)
Parámetros:
- iterable: colección de números (lista, tupla).
- numero: valor a redondear; ndigits: decimales (opcional).
Promedio = sum(lista) / len(lista). Para mostrar 2 decimales: print(f"{promedio:.2f}").
Precisión binaria: 0.1 + 0.2 puede dar 0.30000000000000004. Usá formato con f-strings para salidas prolijas.

Listas (primer vistazo)

colores = ["rojo", "verde", "azul"]
colores.append("violeta")
colores.insert(1, "negro")
print(colores)
lista.append(elemento) | lista.insert(posicion, elemento)
Parámetros:
- append: elemento único al final.
- insert: posición (int, empieza en 0) y elemento a insertar.
Más métodos y slicing estarán en la página dedicada de Listas.

Diccionarios (primer vistazo)

persona = {"nombre": "Ana", "edad": 25}
persona["ciudad"] = "Buenos Aires"
print(persona["nombre"])
diccionario = {clave: valor, ...} | diccionario[clave] = valor | diccionario[clave]
Parámetros:
- clave: identificador (str, int, etc.).
- valor: dato asociado.
Acceso: diccionario[clave] devuelve el valor; asignación crea/actualiza pares clave–valor.

Buen hábito: salida formateada

nombre = "Lucía"; promedio = 13.3333333333
print(f"Hola, {nombre}. Tu promedio es {promedio:.2f}.")
f"texto {variable:formato}"
Formatos útiles: {:.2f} fija 2 decimales; {:d} entero; {:>6} alineación a la derecha con ancho 6.

Tuplas

# Una tupla es como una lista, pero inmutable colores = ("rojo", "verde", "azul") # Acceder al primer elemento print(colores[0]) # Saber cuántos elementos tiene print(len(colores)) # Verificar si un valor está en la tupla print("verde" in colores)
Explicación paso a paso:
1. Creamos una tupla llamada colores con tres elementos.
2. Usamos colores[0] para acceder al primer valor.
3. Con len(colores) obtenemos la cantidad de elementos.
4. Con "verde" in colores verificamos si "verde" está dentro de la tupla.
# Salida en consola rojo 3 True
Una tupla es una colección ordenada de elementos, similar a una lista, pero no se puede modificar (es inmutable). Se usan cuando querés agrupar datos que no deberían cambiar, como coordenadas, fechas o configuraciones fijas.

Ejemplo práctico: coordenadas

# Representar un punto en el plano punto = (10, 20) # Desempaquetar la tupla en dos variables x, y = punto print("X:", x) print("Y:", y)
# Salida en consola X: 10 Y: 20
En este ejemplo, la tupla punto guarda dos valores (x e y). Gracias al desempaquetado, podemos asignar cada valor a una variable en una sola línea.

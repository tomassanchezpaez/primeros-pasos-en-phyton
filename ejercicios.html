Ejercicios en Python: poné en práctica lo aprendido

Inicio Condicionales Bucles Funciones Ejercicios para practicar

Aquí encontrarás una serie de ejercicios para aplicar lo aprendido en Condicionales, Bucles y Funciones. La idea es que intentes resolverlos por tu cuenta antes de ver la solución.

Ejercicio 1: Condicionales

Escribí una función que reciba una lista de números y devuelva el número mayor de ellos.

Ejercicio 2: Bucles

Escribí una función que reciba una lista de números y devuelva una lista con los números pares.

Ejercicio 3: Funciones con parámetros

Escribí una función que reciba una lista de números y devuelva el promedio.

Ejercicio 4: Funciones con bucles

Escribí una función que reciba una lista de números y devuelva una lista con los cuadrados de cada uno.

Ejercicio 5: Desafío

Escribí una función que reciba una lista de números y devuelva una lista con los números primos.

Ejercicio 6: Validación de usuario

Escribí un programa que pida un nombre de usuario y una contraseña. El usuario tiene 3 intentos para ingresar correctamente ambos datos. Si lo logra, mostrar "Acceso concedido". Si falla, mostrar "Acceso denegado".

Ejercicio 7: Números primos

Escribí una función que reciba un número entero y devuelva `True` si es primo, o `False` si no lo es.

Ejercicio 8: Contador de vocales

Escribí una función que reciba una cadena de texto y devuelva cuántas vocales contiene.

Ejercicio 9: Suma acumulada

Escribí una función que reciba una lista de números y devuelva una nueva lista con la suma acumulada. Ejemplo: `[1, 2, 3] → [1, 3, 6]`

Ejercicio 10: Palíndromos

Escribí una función que reciba una palabra y devuelva `True` si es un palíndromo (se lee igual al revés).

Ejercicio 11: Conversión de temperatura

Escribí una función que reciba una temperatura en grados Celsius y la convierta a Fahrenheit.

Ejercicio 12: Números pares e impares

Escribí una función que reciba una lista de números y devuelva dos listas: una con los pares y otra con los impares.

Ejercicio 13: Contador de palabras

Escribí una función que reciba una frase y devuelva cuántas palabras contiene.

Ejercicio 14: Factorial

Escribí una función que reciba un número entero y devuelva su factorial.

Ejercicio 15: Verificador de contraseña segura

Escribí una función que reciba una contraseña y verifique si es segura. Una contraseña segura debe tener al menos 8 caracteres, incluir una mayúscula, una minúscula y un número.

Ejercicio 16: Suma de múltiplos

Escribí una función que reciba un número entero positivo `n` y devuelva la suma de todos los múltiplos de 3 o 5 menores a `n`.

Ejercicio 17: Filtrar por longitud

Escribí una función que reciba una lista de palabras y un número `n`, y devuelva una nueva lista con las palabras que tienen más de `n` caracteres.

Ejercicio 18: Simulación de cajero

Escribí un programa que simule un cajero automático. El usuario puede consultar saldo, depositar o retirar dinero. El saldo inicial es 0. El programa debe mostrar un menú y permitir operaciones hasta que el usuario elija salir.

Ejercicio 19: Reemplazo de caracteres

Escribí una función que reciba una cadena de texto y reemplace todas las vocales por asteriscos `*`. Ejemplo: `"Hola mundo"` → `"H*l* m*nd*"`

Ejercicio 20: Verificador de orden

Escribí una función que reciba una lista de números y devuelva `True` si están ordenados de menor a mayor, o `False` si no lo están.
